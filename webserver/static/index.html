Fána

Fána is an open-source feature flag management platform that facilitates targeted application releases to progressively larger qualified subsets of their user base.

By providing developers the means to gradually turn features "on" for specific audiences, Fána is a tool with which to control testing in production safely and responsibly.

Built using Go, Node.js, React, Docker, PostgreSQL, with automated deployment to AWS, Fána is easy to set up and integrate with your applications.
